# lab68bio

**Build your bio & portfolio — share it at `username.bio.lab68`.**

lab68bio is a drag-and-drop bio and portfolio builder that lets anyone create a beautiful personal page in minutes. Pick from pre-built components (hero, about me, social links, portfolio grid, experience, skills, contact form, and more), arrange them visually, customise the theme, and publish with one click.

## Features

- **Drag-and-drop editor** — no code required
- **Pre-built components** — hero banner, about me, social links, portfolio grid, experience, skills, contact form, resume
- **Custom themes** — personalise colours, fonts, and layout
- **Instant publishing** — go live at `username.bio.lab68`
- **Auth & dashboard** — register, login, and manage multiple pages

## Tech Stack

- **Next.js 16** (App Router)
- **React 19**
- **Tailwind CSS 4**
- **Drizzle ORM + Neon Postgres**
- **@dnd-kit** for drag-and-drop
- **jose** for JWT auth

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the landing page.
